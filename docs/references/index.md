# References

This directory is for supporting material that is informative but not the
primary source of truth.

## Reference Policy

- Canonical repo rules belong in `AGENTS.md`, `ARCHITECTURE.md`, and the
  normative docs in `docs/`.
- External articles, copied snippets, and implementation notes belong here when
  they are worth retaining.
- Do not treat references as higher priority than repo-local canonical docs.

## Current References

- `modal-sandbox.md`: local quickstart for launching a reusable Modal sandbox
  from this repo.
- `modal-openai-agents-sdk-blog.md`: applied notes from Modal's blog on
  building an OpenAI Agents SDK harness on Modal Sandboxes.
